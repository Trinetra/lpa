# Fonts

Noto Sans / Noto Serif, downloaded from Google Fonts. Licensed under the
SIL Open Font License 1.1 (free to embed/redistribute).

Embedded here (rather than relying on ReportLab's built-in Helvetica/Times)
because those built-in fonts don't include the ₹ (rupee) glyph — it
rendered as a missing-glyph box otherwise. Registered in `services/pdf.py`.
